# AI+X 창업 해커톤 우수상 수상작

폐 CT 영상 데이터를 기반으로 의심 병변을 탐지하고, 웹 기반 UI를 통해 누구나 쉽게 사용할 수 있도록 구현한 AI 의료영상 분석 프로젝트입니다.

본 프로젝트는 **AI+X 창업 해커톤 우수상 수상작**입니다.

---

## 프로젝트 개요

본 프로젝트는 지역 의료 인력 부족과 영상의학 전문의 부족 문제를 해결하기 위해 기획되었습니다.

국내 영상검사 수요는 지속적으로 증가하고 있지만, 이를 판독할 영상의학 전문의 수는 충분히 증가하지 못하고 있습니다. 특히 지방의료원과 지역 병원에서는 전문의 부족으로 인해 영상 판독 지연과 의료 접근성 저하 문제가 발생할 수 있습니다.

이를 해결하기 위해 AI허브의 폐 CT 데이터 약 1만 건을 활용하여 YOLOv8 기반 객체 탐지 모델을 학습하였고, 폐 CT 이미지에서 정상 조직과 의심 병변을 빠르게 1차 탐지할 수 있는 시스템을 개발했습니다.

실제 테스트 결과, 이미지 1장당 약 2초 내외로 분석이 가능했으며, 이를 통해 의사의 초진 부담을 줄이고 의료진의 판독 보조 도구로 활용될 가능성을 확인했습니다.

---

## 지원동기 및 문제 정의

### 사회적 배경

- 지방의료원 인력 부족과 수도권 의사 쏠림 현상이 심화되고 있습니다.
- 서울 일부 상급종합병원은 전문의 충원이 이루어지고 있으나, 지역 다수 병원은 의료 인력 부족이 심각한 상황입니다.
- 국내 영상검사는 연평균 약 7% 증가하고 있지만, 영상의학과 전문의는 연평균 약 3% 증가에 그치고 있습니다.
- 2013년 영상의학과 전문의 수는 3,065명이었으나, 2023년에는 4,206명으로 증가했습니다. 같은 기간 영상검사 수요는 크게 증가했지만 전문의 수 증가는 이에 미치지 못했습니다.

### 해결하고자 한 문제

영상검사 증가에 비해 전문 판독 인력이 부족한 상황에서, AI가 폐 CT 영상을 1차로 분석하여 의심 병변을 빠르게 탐지한다면 의료진의 업무 부담을 줄이고 진단 보조 효율성을 높일 수 있습니다.

## 모델 학습 과정

본 프로젝트에서는 AI허브 폐 CT 데이터 약 1만 건을 활용하여 YOLOv8 모델을 학습했습니다.

학습에는 `Ultralytics YOLOv8m` 모델을 사용했으며, 이미지 크기는 `640`, 학습 에포크는 `80`, 배치 크기는 GPU 환경에 따라 자동 조정되도록 구성했습니다.

```python
from ultralytics import YOLO

model = YOLO("yolov8m.pt")

results = model.train(
    data="data.yaml",
    imgsz=640,
    epochs=80,
    batch=8,
    device=0,
    optimizer="auto",
    cos_lr=True,
    patience=30
)
